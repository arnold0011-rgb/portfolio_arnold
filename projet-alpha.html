Projet Alpha

Développement Front-end moderne

À propos du projet

Le Projet Alpha est une interface utilisateur conçue pour la gestion de données en temps réel. L'objectif était de rendre des données complexes simples à lire pour l'humain.

Technologies utilisées :

  • HTML5 / CSS3 (Flexbox & Grid)
  • JavaScript (ES6+)
  • API de données météo
